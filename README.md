## 📖 Deskripsi

Aplikasi manajemen perpustakaan berbasis Laravel yang dibuat untuk **Uji Kompetensi Keahlian (UKK) SMK RPL**.
Aplikasi ini memudahkan pengelolaan data buku, siswa, serta proses peminjaman dan pengembalian.

---

## 🚀 Fitur Utama

* 🔐 Login & Register
* 📊 Dashboard Statistik
* 📚 Manajemen Data Buku
* 👨‍🎓 Data Siswa
* 🔄 Peminjaman & Pengembalian
* 📈 Laporan Data

---

## 🛠️ Teknologi yang Digunakan

* Laravel
* PHP
* MySQL
* Bootstrap
* JavaScript

---

## ⚙️ Cara Menjalankan Project

```bash
git clone https://github.com/alfinmuzakkiiman/Uji-Kompetensi-Keahlian-Smk.git
cd Uji-Kompetensi-Keahlian-Smk
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
```
